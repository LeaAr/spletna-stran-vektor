Kaj sploh je vektorska grafika?

Vektorska grafika je v računalniški grafiki način predstavitve slik ki, bazira na matematičnih izračunih vektorjev različnih oblik. Oblike objektov na sliki so določene z različnimi osnovnimi geometrijskimi objekti, kot so: točke, črte, krivulje in mnogokotniki. Vsak objekt je v pomnilniku določen z nizom matematičnih enačb. Program shrani podatke, kot so barva podlage, barva črte, debelina črte, barva polnila, oblika črte.

Kot vsaka stvar ima tudi vektorska grafika svoje prednosti in svoje slabosti.

Prednosti:

  • neodvisnost od ločljivosti slike
  • vsaka komponenta definirana posebej
  • možnost povezav do delov slike
  • majhna velikost datotek
  • gladke krivulje
  • učinkovitost

Slabosti:

  • omejenost detajlov
  • omejenost učinkov
  • nenaravnega videza
  • težave pri tiskanju

Formati

Vektorski formati se lahko med seboj zelo razlikujejo. Nekateri zapišejo le nekaj preprostih ukazov, drugi pa so lahko mnogo bolj kompleksni. Vsi formati pa si lastijo nekaj skupnih značilnosti. Mednje spada način kodiranja podatkov in način dela z barvami. Navedenih je le nekaj primerov, a najpomembnejši je SVG.

Kratica SVG pomeni Scalable Vector Graphics, kar v slovenščini pomeni umerljiva vektorska grafika. Je format ki omogoča shranjevanje in prikaz 2D slik. Prednosti SVG formata so da je manjše datoteke, je neodvisen od resolucije, tekstovne napise in opise se lahko išče z iskalnikom, možna je povezava do delov slike. Predvaja tudi animacijo.

    Ostali formati:
  • EPS - vsestranski format
  • CDR - format za CorelDraw
  • SWF - format za gledanje animacije (FlashPlayer)
  • AI - format programa Adobe IllustratorPDF
